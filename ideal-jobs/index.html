Blockchain Developer

Shared Components of All the Ideal Jobs:

  • Writing skills and verbal communication
  • Technical skills
  • Time management

Differences Between the Ideal Jobs:

  • A CSO analyst tests for vulnerabilities, analyses risks, breaches, data (to suggest changes), and combat malware.

Jesse Catchpole

Cyber Security Operations Analyst

Jesse's ideal job would be working as a Cyber Security Operations Analyst. This job primarily entails working with management and operations teams to safeguard IT systems from intrusion, unwanted traversal and providing a safer environment for private data. This role stretches outside of the virtual world of computers and into the physical world, where people are as big a vulnerability as leaving your outward-facing networks freely accessible to the internet.


There are several common elements between each of the ideal jobs stated here. First, as these jobs include an element of computing, there will be a need for a number of technical abilities. First of which is an understanding of how each job implements their computer system. This would include the OS, the computer hardware, networking protocols, the day-to-day troubleshooting of these computers and the ability tooperate these systems in a safe and secure manner.


As with many jobs, the ability to communicate effectively and operate coherently within a team environment is a must. Alongside this, employees will need to be able to think critically, be adaptable to new technologies and manage their own time competently to achieve the goals their positions require.


There is some differentiation to be made between each job. As their specialties vary, focus is required in different fields. As a Blockchain Developer competency will be required across several programming languages, such as C++, Java, Python and JavaScript. This role will require the ability to customise their code so that performance of the blockchain is scalable and won't result in a slow system when the size of the network increases.


To be a CSO Analyst, you will need to not only implement security-specific software, but also understand the policies and procedures as stipulated by the government or the organisation itself. A strong understanding of network logs and implementing software changes that secure your network against intrusion is key. In the physical world, you will also need to know how to handle people, as they are an avenue of infiltration for phishers.


To be a Construction Project Manager,a good grasp of the technical applications involved with logistics and design of the construction environment is needed to allow the project to work to completion smoothly.


To be a Blast Engineer, it will require a grasp on mining-specific programs like Aegis, Datamine 5DP/UG and Ventsim. All these aspects come together to create a safe environment both before and after blasting has occurred.


These careers would require extensive training in IT, at least a bachelor's degree in most cases. This is to give a solid foundation that will support the specialisation that comes with each job.


Where they will differ, however, is that the Blast Engineer will require a tertiary education in mine engineering. The Security Analyst will need security-specific education (CEH, CISM, CISSP). The Blockchain Developer will need knowledge on implementing blockchain frameworks and how these fit into large-scale production applications. The Construction Project Manager will need to understand the use of industry-specific software like Jobpac, as well as knowledge on how to impact the environment the least.
